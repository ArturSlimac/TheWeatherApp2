TopSearchBar

@Composable
fun TopSearchBar(searchText: String, onSearchTextChange: (String) -> Unit, isSearching: Boolean, onToggleSearch: () -> Unit, foundCitiesState: Response<CityModel>, onCitySelected: (CityItemModel) -> Unit)

A composable function that displays a top search bar with the ability to search and select cities.

The search bar allows users to input a search query, triggers a search action, and displays a list of found cities based on the search query. The UI changes dynamically based on the state of the search and the results.

Parameters

searchText

The current text in the search bar.

onSearchTextChange

Callback function to handle changes to the search text.

isSearching

A boolean indicating whether the search bar is active.

onToggleSearch

Callback function to toggle the search bar's active state.

foundCitiesState

The current state of the search results, represented by a Response of CityModel.

onCitySelected

Callback function triggered when a city is selected from the search results.
