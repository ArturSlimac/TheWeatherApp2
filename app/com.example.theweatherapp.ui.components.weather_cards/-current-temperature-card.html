CurrentTemperatureCard

@Composable
fun SharedTransitionScope.CurrentTemperatureCard(animatedVisibilityScope: AnimatedVisibilityScope? = null, currentTemperatureItem: CurrentTemperatureItem, key: String? = null)

A composable function that displays the current temperature in a card with an animated gradient background. The card supports shared element transitions if animatedVisibilityScope is provided.

Parameters

animatedVisibilityScope

The scope AnimatedVisibilityScope used to handle shared element transitions. If null, no transition is applied.

currentTemperatureItem

The CurrentTemperatureItem data model containing the current temperature and UI details.

key

An optional key used to identify the shared element for transition animations.
